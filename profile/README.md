## Overview

Slidepad Advanced is a slide-over panel application that provides quick access to web applications, bookmarks, and online services in a floating side panel that appears over your current work without switching to a full browser or losing context. It handles the common need to quickly check a reference, respond to a message, look up information, or interact with a web service without the full context switch that opening a browser window requires.

Slide-over panel appears from the screen edge with a keyboard shortcut or mouse gesture, displaying web content in a focused panel alongside the current application. The panel can display any website or web application in a contained browser view. Multiple tabs within the panel organize different web services for quick switching.

Web app bookmarks provide one-click access to frequently used web services — email, messaging, project management, documentation, social media — in the slide-over panel without navigating through a full browser. The bookmarks function as mini-applications that load instantly when the panel opens.

Floating and pinned modes either float the panel over the current application or pin it alongside, creating a persistent side panel for reference material during focused work. The panel width is adjustable, and the panel can be positioned on either screen edge.

For users who frequently check web services during focused work and want quick access without the context switch of opening a full browser, Slidepad Advanced is the panel tool that keeps web resources one gesture away.

## Performance

Slidepad balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
